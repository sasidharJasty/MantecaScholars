# Manteca Scholars

Manteca Scholars is a non-profit organization that coordinates academic competitions and programs for Sierra High School. Founded to inspire excellence above all else.

## Getting Started

1. **Install Dependencies**
   ```sh
   npm install
   ```

2. **Run Development Server**
   ```sh
   npm run dev
   ```

## Technology Stack

- **Framework**: React + Vite
- **Styling**: Tailwind CSS + shadcn/ui
- **Database**: Supabase

# Step 3: Install the necessary dependencies.
npm i

# Step 4: Start the development server with auto-reloading and an instant preview.
npm run dev
